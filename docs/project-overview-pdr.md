# Project Overview - PMS (Project Management System)

## Tổng quan

PMS là hệ thống quản lý dự án giúp các team theo dõi và quản lý công việc hiệu quả.

## Mục tiêu

1. Tạo nền tảng quản lý dự án dễ sử dụng
2. Hỗ trợ collaboration giữa các thành viên
3. Theo dõi tiến độ real-time
4. Tạo báo cáo tự động

## Tính năng chính

### 1. Quản lý Dự án
- Tạo/sửa/xóa dự án
- Phân quyền truy cập
- Template dự án

### 2. Quản lý Tasks
- Kanban board
- Task dependencies
- Deadline và reminders
- Subtasks

### 3. Team Management
- Invite members
- Roles & permissions
- Workload view

### 4. Reporting
- Progress charts
- Time tracking
- Export reports

## Tech Stack (đề xuất)

- Frontend: Next.js + TypeScript + TailwindCSS
- Backend: NestJS/Express
- Database: PostgreSQL
- Cache: Redis
- Auth: JWT + OAuth2

## Timeline

- Phase 1: Core features (Projects, Tasks, Users)
- Phase 2: Advanced features (Reporting, Integrations)
- Phase 3: Mobile app
