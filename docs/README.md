## System Architecture

**Frontend:**
- Next.js 14 with TypeScript
- TailwindCSS + shadcn/ui
- React Query for data fetching

**Backend:**
- NestJS with TypeScript
- PostgreSQL database
- Redis caching
- MinIO object storage

**Deployment:**
- Docker & Docker Compose
- Nginx reverse proxy
- Let's Encrypt SSL
